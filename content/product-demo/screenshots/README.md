# Product Demo Screenshots (Local Draft)

Drop screenshots in this folder (top-level) to test them on the local draft page carousel.

Suggested filenames used by the sample page:

- `dashboard.png`
- `live-session.png`
- `ai-chat.png`

Only top-level files in `content/product-demo/screenshots/` are shown in the carousel.
Nested folders are ignored.

This folder is part of a **draft page bundle** (`content/product-demo/index.md` with `draft: true`).

- Local preview: run `hugo serve -D`
- Production build/deploy: draft content is excluded by default
